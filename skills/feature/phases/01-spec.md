# Phase 1: SPEC

## Goal
Define types, API contracts, and task cards that builders will implement.

## Agent
Delegate to the `spec-writer` agent with the feature name and in-scope layers.

## Steps
1. Define shared types in `shared/types/`
2. Define the API contract (request/response schemas, endpoints)
3. Create task cards — one per builder, describing what to build and test

## Completion
- Types defined and exported
- API contract documented
- Task cards ready for each in-scope builder
- Mark spec task as completed, move to BUILD phase
